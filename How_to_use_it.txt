
How to use scripts

1) start cmd and ssh session (ssh username@doom30.metacentrum.cz) or use putty.
2) run ./initHuygens.sh
3) start your VNC client with provided informations
4) start xterm on your remote desktop and run ./hupro.sh
5) logout properly after you finish. Gui will be shut down when your reservation time expire.

LINUX users:
download and start ./dekonvoluce.sh in your local terminal. Everything will be done for you automatically till start ssvnc. You can use SHIFT+INSERT to paste a vnc login from the clipboard.

Script needs to be adjusted in case
1] you want to use different VNC viewer
2] you log in with ssh-key to metacentrum servers

dekonvoluce.sh still needs to be rewriten to dekonvoluce.bat script for windows users.
